# 📸 ScrCapture Pro 사용 설명서

ScrCapture Pro는 클라우드(VDI), 듀얼 모니터 등 복잡한 업무 환경에서도 가장 빠르고 정확하게 화면을 캡처하고 녹화할 수 있도록 설계된 **전문가용 캡처 리모컨**입니다.

---

## 다운로드

https://github.com/ilove8790/ScreenCapture/ScreenCapture.zip


## 설치

다운로드 받은 압축 파일을 원하는 위치에 폴더명(ScreenCapture)을 생성하고 압축을 풀어 놓으면 됩니다.
실행중 필요한 폴더는 그 아래에 자동으로 생성됩니다.


## 📌 단축키 전체 목록 (Global Hotkey)

> 프로그램 창이 숨겨져 있거나 다른 창 작업 중에도 **모든 단축키는 항상 작동**합니다.

| 단축키 | 기능 | 결과 저장 |
|---|---|---|
| `Ctrl + 0` | 리모컨 창 표시/숨김 토글 | — |
| `Ctrl + 1` | **전체 화면** 캡처 | 클립보드 |
| `Ctrl + 2` | **현재 활성화된 창** 캡처 | 클립보드 |
| `Ctrl + 3` | **영역 선택** 모드 진입 / 선택 확정 | 클립보드 |
| `Ctrl + 좌클릭` | **고정 영역 즉시 재캡처** (빨간 테두리 영역) | 클립보드 |
| `Ctrl + 4` | 마지막 캡처 이미지를 **PNG 파일로 저장** | `Screenshots/` 폴더 |
| `Ctrl + 5` | 고정 영역 **MP4 녹화** 시작/정지 | `Recordings/` 폴더 |
| `Ctrl + 6` | 고정 영역 **GIF 녹화** 시작/정지 | `Recordings/` 폴더 |

## 🔷 기능 상세 설명

### 1. Ctrl + 1 — 전체 화면 캡처
전체 화면(다중 모니터 포함)을 한 번에 캡처하여 클립보드에 복사합니다.
캡처 직후 `Ctrl+V`로 어디든 즉시 붙여넣기 가능합니다.

### 2. Ctrl + 2 — 활성화 창 캡처
현재 포커스가 맞춰진 창의 영역만 정밀하게 오려 클립보드에 복사합니다.

### 3. Ctrl + 3 — 영역 선택 (빨간 테두리 설정)

영역 선택은 **두 단계**로 동작합니다.

**① 첫 번째 Ctrl+3** — 영역 선택 화면 진입
- 화면이 어두워지며 전체 화면 위에 선택 오버레이가 나타납니다.
- 마우스로 드래그하여 캡처할 사각 영역을 지정합니다.
- 이전에 선택한 영역이 있으면 그 영역이 복원되어 가장자리/모서리를 잡아 바로 수정할 수 있습니다.
- 내부를 드래그하면 영역 전체 이동, 가장자리/모서리를 드래그하면 크기 조절이 가능합니다.

**② 두 번째 Ctrl+3 또는 Enter** — 캡처 확정
- 선택한 영역이 즉시 캡처되어 클립보드에 복사됩니다.
- 화면에 **빨간색 테두리 가이드라인**이 남아 고정 영역을 표시합니다.

> **ESC** 키를 누르면 캡처를 취소하고 선택 화면을 닫습니다.

---

#### 🔴 빨간색 테두리 해제 방법

`Ctrl+3`으로 영역을 확정하면 화면에 빨간 테두리가 남습니다.
아래 방법으로 해제할 수 있습니다.

| 방법 | 설명 |
|---|---|
| `Ctrl+3` 다시 누르기 | 영역 선택 화면 진입 → **ESC** 로 취소하면 테두리가 사라짐 |
| `Ctrl+3` → 영역 밖 클릭 후 Enter | 빈 영역(크기 0)을 확정하면 테두리 자동 해제 |
| 프로그램 재시작 | 프로그램을 종료 후 다시 실행 |

> **가장 빠른 해제:** `Ctrl+3` 누름 → ESC 키

---

### 4. Ctrl + 좌클릭 — 고정 영역 반복 캡처
`Ctrl+3`으로 잡아둔 **빨간 테두리 영역**을 `Ctrl` 키를 누른 채 마우스를 클릭할 때마다 즉시 재캡처합니다.
PPT 슬라이드나 웹페이지를 넘기면서 같은 위치를 연속으로 찍을 때 매우 유용합니다.

> 빨간 테두리가 설정되지 않은 상태에서는 작동하지 않습니다.

### 5. Ctrl + 4 — 이미지 파일 저장 (PNG)
가장 마지막에 캡처한 이미지를 `Screenshots/capture_YYYYMMDD_HHMMSS.png` 형식으로 저장합니다.

> 캡처를 한 번도 하지 않은 상태에서는 "❌ 저장할 이미지 없음" 메시지가 표시됩니다.

### 6. Ctrl + 5 / 6 — 화면 녹화 (MP4 / GIF)
- **Ctrl+3**으로 고정 영역을 먼저 설정한 뒤 사용해야 합니다.
- 한 번 누르면 🔴 녹화 시작, 다시 누르면 자동 저장 및 정지됩니다.
- 빨간 테두리 안쪽(테두리 두께 5px 제외)만 깔끔하게 녹화됩니다.

| 단축키 | 형식 | 저장 폴더 |
|---|---|---|
| `Ctrl + 5` | MP4 동영상 | `Recordings/` |
| `Ctrl + 6` | GIF 애니메이션 | `Recordings/` |

## ⚙️ 기타 기능

### 녹화 FPS 설정
리모컨의 **[녹화 프레임]** 버튼을 클릭하여 10 FPS ↔ 30 FPS를 전환합니다.
- **10 FPS (기본)**: 파일 용량 최소화, 문서/메신저 첨부에 적합
- **30 FPS**: 부드러운 움직임이 필요한 고화질 영상

### 창 숨기기 / 다시 열기
- 타이틀바의 **`—`** 버튼 또는 **Ctrl+0** 으로 리모컨 창을 숨깁니다.
- 숨긴 상태에서도 모든 단축키는 정상 작동합니다.
- **Ctrl+0** 을 다시 누르면 창이 다시 나타납니다.

## 🚀 특장점

1. **완벽한 클립보드 호환 (VDI/원격 최적화)**
   캡처된 이미지는 최적화된 비트맵(DIB) 포맷으로 클립보드에 복사됩니다. 사내 메신저, 원격 데스크톱(VDI), 엑셀/파워포인트 등 어디서든 `Ctrl+V`로 원본 화질 그대로 붙여넣기 가능합니다.

2. **스마트 영역 수정**
   `Ctrl+3`을 다시 누르면 이전에 잡아둔 영역이 그대로 복원되어 크기/위치를 미세 조정할 수 있습니다.

3. **깔끔한 화면 녹화**
   빨간 테두리는 영상 결과물에 포함되지 않습니다. 테두리 두께(5px)만큼 안쪽을 자동 크롭하여 완전히 깔끔한 영상만 저장합니다.

4. **안전한 좌클릭 캡처**
   반드시 `Ctrl`을 누른 상태에서만 클릭 캡처가 작동하므로 평상시 업무 중 실수로 발동하지 않습니다.

5. **완벽한 스텔스 모드**
   `Ctrl+0` 또는 `—` 버튼으로 창을 완전히 숨긴 채 단축키만으로 모든 기능을 사용할 수 있습니다.
